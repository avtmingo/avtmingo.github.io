  • All Componets/Files

    • Folder BuildEnv - Docker File

      Docker Basically gives us the facility of creating a build environmnet so that we can setup our Operating System.It Uses the concept of docker image .

      To build a docker image we use the following command :

      docker build buildenv -t myos-buildenv


      To create an instance of the image we use the command:

      docker run --rm -it -v "${pwd}:/root/env" myos-buildenv
